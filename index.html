Como o código chega até a internet

Commit, Push e Deploy

Commit

Salvar uma versão do seu projeto. Como tirar uma foto do código naquele momento. Você pode voltar atrás se algo der errado.

Push

Enviar para a nuvem. O commit salva no seu computador. O push manda pro GitHub — seguro, acessível de qualquer lugar.

Deploy

Publicar na internet. Quando você faz deploy, seu site fica acessível para qualquer pessoa no mundo.

Commit → Push → Deploy. Você faz no computador, manda pro GitHub, e publica na Vercel.

A memória da IA

CLAUDE.md e Memória

O Claude Code aprende sobre o seu projeto e lembra entre sessões.

CLAUDE.md

Um arquivo de texto que você coloca na pasta do projeto. O Claude lê ele sempre que inicia. Ali você escreve as regras: “use Tailwind”, “escreva em português”, “o banco é Supabase”.

Memória automática

O Claude salva aprendizados sozinho: comandos de build, bugs que resolveu, decisões que tomou. Na próxima sessão, ele já sabe o que fazer.

# Exemplo de CLAUDE.md
Este projeto é um site para a PLAYMIX.
Use Next.js, Tailwind e TypeScript.
O banco de dados é Supabase.
Escreva tudo em português.

A dupla perfeita

Claude Code + Cursor — como funcionam juntos

Claude Code

Você conversa e a IA cria tudo do zero. Funciona no terminal.

Cursor

Você vê os arquivos e edita visualmente. É o editor de código.

Juntos

Claude Code cria, Cursor ajusta. Pode usar o Code sozinho no terminal OU dentro do Cursor.

O Claude Code funciona dentro do Cursor como plugin (= extensão que você instala dentro do editor). Melhor dos dois mundos.

O que existe pra construir

Tecnologias — qual usar?

Cada tecnologia serve pra um tipo de projeto. Você não precisa escolher — a IA escolhe pra você. Mas é bom saber o que existe.

HTML + CSS + JavaScript

O básico da web. Sites simples, páginas estáticas, portfólios. Abre direto no navegador.

→ Site simples, landing page

Next.js (React)

O padrão profissional. Sites dinâmicos, apps web, e-commerce. O que vamos usar hoje.

→ Site profissional, aplicativo web

React Native / Expo

Aplicativos para celular (iOS e Android) usando a mesma linguagem da web.

→ App para iPhone e Android

Swift / Kotlin

Apps nativos — Swift para iPhone, Kotlin para Android. Máxima performance.

→ App nativo de alta performance

Python

Automações, scripts, inteligência artificial, análise de dados. Não é pra sites.

→ Automação, IA, dados

Flutter

Apps para celular, desktop e web com uma única base de código. Do Google.

→ App multiplataforma

Instalação

Como instalar o Claude Code

Copie a linha inteira abaixo e cole no terminal. Aperte Enter.
O $ indica onde digitar — não digite o $. Para colar: Mac = Cmd+V, Windows = Ctrl+V ou clique direito.
# No Mac (abra o Terminal)
$ curl -fsSL https://claude.ai/install.sh | bash

# No Windows (abra o PowerShell)
$ irm https://claude.ai/install.ps1 | iex

Vai aparecer texto rodando na tela. É normal. Espere até pedir login.
Precisa de conta Claude Pro ($20/mês). O plano gratuito não inclui o Code.
