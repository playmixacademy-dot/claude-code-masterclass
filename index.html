Claude Code10PLAYMIX
Instalação

Como instalar o Claude Code

Copie a linha inteira abaixo e cole no terminal. Aperte Enter.
O $ indica onde digitar — não digite o $. Para colar: Mac = Cmd+V, Windows = Ctrl+V ou clique direito.
# No Mac (abra o Terminal)
$ curl -fsSL https://claude.ai/install.sh | bash

# No Windows (abra o PowerShell)
$ irm https://claude.ai/install.ps1 | iex

Vai aparecer texto rodando na tela. É normal. Espere até pedir login.
Precisa de conta Claude Pro ($20/mês). O plano gratuito não inclui o Code.

O que existe pra construir

Tecnologias — qual usar?

Cada tecnologia serve pra um tipo de projeto. Você não precisa escolher — a IA escolhe pra você. Mas é bom saber o que existe.

HTML + CSS + JavaScript

O básico da web. Sites simples, páginas estáticas, portfólios. Abre direto no navegador.

→ Site simples, landing page

Next.js (React)

O padrão profissional. Sites dinâmicos, apps web, e-commerce. O que vamos usar hoje.

→ Site profissional, aplicativo web

React Native / Expo

Aplicativos para celular (iOS e Android) usando a mesma linguagem da web.

→ App para iPhone e Android

Swift / Kotlin

Apps nativos — Swift para iPhone, Kotlin para Android. Máxima performance.

→ App nativo de alta performance

Python

Automações, scripts, inteligência artificial, análise de dados. Não é pra sites.

→ Automação, IA, dados

Flutter

Apps para celular, desktop e web com uma única base de código. Do Google.

→ App multiplataforma

Prática25PLAYMIX
Prompt = o pedido que você faz para a IA. Quanto mais detalhado, melhor.

O pedido que vamos usar

"Crie um site profissional completo com Next.js para
a PLAYMIX, uma produtora audiovisual especializada em
filmes corporativos, eventos e conteúdo para redes sociais.

Estrutura do site:
1. Tela inicial cinematográfica com vídeo de fundo em loop
2. Seção "Sobre nós" com história da produtora
3. Portfólio com galeria de vídeos e fotos em grid
4. Serviços: filmagem, edição, drone, fotografia,
motion graphics — cada um com ícone e descrição
5. Formulário de orçamento com: nome, email, telefone,
tipo de serviço (dropdown) e descrição do projeto
6. Rodapé com links para Instagram, YouTube e WhatsApp

Design: fundo escuro (#0a0a0a), texto branco, detalhes
em laranja (#E8590C). Visual cinematográfico e premium.
100% responsivo (celular, tablet, desktop).
Animações suaves ao rolar a página com Framer Motion.
Use Next.js App Router, Tailwind CSS e TypeScript."
A técnica que faz diferença

Como pedir bem pra IA

Funciona

"Crie um cabeçalho escuro com logo à esquerda"

"Use a paleta de cores do site X como referência"

"Primeiro faça a tela inicial, depois a galeria"

"O menu não abre no celular, corrija"

Não funciona

"Faz um site bonito" — bonito como?

"Melhora isso" — melhorar o que?

"Faz tudo de uma vez" — confunde a IA

Aceitar sem testar no celular
